## Repository Structure

The repository is organized by assignment, typically following this naming convention:

*   `ex_XX.tex`: The LaTeX source code for the solutions to Assignment `XX`.
*   `templete.tex`: The LaTeX template used for formatting the assignments.
*   `*.pdf`: Problem sheets and compiled solution documents.

## Usage

To compile the solution for a specific assignment (e.g., Assignment 05), ensure you have a LaTeX distribution installed (like TeX Live or MiKTeX) and run the following command in your terminal:

```bash
pdflatex ex_05.tex
```

Replace `ex_05.tex` with the specific assignment file you wish to compile.
